# cpp00

Questo modulo introduce i concetti base della programmazione orientata agli oggetti in C++.

## Esercizi

### ex00
Implementazione di una classe semplice con costruttore, distruttore e metodi di base. Si impara la sintassi delle classi e la gestione della memoria.

### ex01
Gestione di classi con membri privati e pubblici, metodi getter/setter e prime nozioni di incapsulamento.

## Come usare
Ogni sottocartella contiene un esercizio indipendente. Consulta i README interni per dettagli specifici.
